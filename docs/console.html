Console commands

Press ")" under linux and "`" or "²" under windows to show/hide console while playing.
(n is an integer, f a floating point number)


1. fps_on : show the frame rate(frame per sec.)

2. fps_off : hide the frame rate

3. spawn name_of_unit player_id nb_unit : create specifyed unit(s) for the player. Usage:
name_of_unit - specify type of unit by it`s name, usefull for units testing ("any" = random)
player_id - specify unit side (0 - player, 1 - computer)
nb_unit - number of units to spawn

4. shoot : make a screen shot

5. show model : show/hide information about the 3d model of the selected unit

6. timefactor f : set the game speed (1 - is normal)

7. metal : activate/deactivate cheat code for full metal

8. energy : activate/deactivate cheat code for full energy

9. internal name : show/hide internal names of units

10. unitinfo : print information about selected units in the shell

11. exit : leave the game

12. wireframe : activate/deactivate the wireframe mode

13. priority n : 0, 1 or 2, define the program priority(0:low, 1:medium, 2:high)

14. shadow quality n : quality of shadows (n=1: normal, more n is high more the shadows are nice but the game will run slower)

15. shadow ray f : shadow diffusion (need a high enough shadow quality)

16. shadow : activate/deactivate shadows

17. particle : show/hide particles

18. trees : show/hide some features(trees,...)

19. show script : show/hide scripts linked to the selected unit

20. rotate light : activate/deactivate the rotating light

21. freecam : activate/deactivate the free camera mode

22. fps_limit n : set a frame rate limit (n<=0 : no limit)

23. addhp n : add n hit point to selected units (n can be negative)

24. reset_script : stop all running scripts of selected units

25. kill : kill selected units

26. script n : launch the script number n on each selected unit

27. ai debug : show information about A.I. for debugging

28. change state : change musical background

29. video shoot : activate/deactivate video recording

30. zshoot : make a shot of the zbuffer
